# AnimationSoftware

A video editing and motion design software made in C#.

## Main features
* Layers and groups
* Video effects
* C# expressions ( a bit like scripting )
* Video export
* Keyframes and easing
* Undo system

## Technologies used:
* Dear ImGui
* FFMPEG
* OpenGL
